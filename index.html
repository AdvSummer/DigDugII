Desenvolvimento

Foi utilizado Modern OpenGL(OpenGL 3.3+) para o desenvolvimento. Por conta disso, foi necessário desenvolver uma boa base OpenGL antes de poder a começar a trabalhar no jogo de fato, como escrever Shaders e criar estruturas de Mesh e Model para importar OBJs.

Bibliotecas

  • SDL2: Tratamento principal de janelas e contexto OpenGL
  • GLEW: Detecta as extensões OpenGL suportadas pelo sistema
  • GLM: Cálculo com vetores e matrizes
  • FreeImage: Usado para carregar imagens e texturas
  • ASSIMP: Importa vários formatos de modelos 3D

Manual

O objetivo do jogo é eliminar todos os inimigos, para fazer isso, é possível criar rachaduras no chão a partir de buracos que desmoronam o terreno embaixo do inimigo ou empurrar o inimigo para fora do terreno.

Controles

  • W: andar pra cima
  • A: andar pra esquerda
  • S: andar pra baixo
  • D: andar pra direita
  • Espaço: criar rachadura a partir de um buraco
  • F: empurrar inimigo
  • V: alterna tipo de câmera
